# Tracks in the Dark

Welcome to the Tracks in the Dark repository.
You are currently looking at the root of the repository.
From here, you can navigate to the various subdirectories and files that will help you during the hackahton.


## Directory Structure

The repository is organized into multiple directories, each serving a specific purpose.
The main directories are listed below.
Each directory contains a README.md file that provides more detailed information about its contents and how to use it.

- `data/` contains data files that you may need for your project.
- `lecture/` contains materials and resources from the lecture.
